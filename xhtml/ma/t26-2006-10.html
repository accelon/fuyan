（四二）[4]《中阿含》[5]〈習相應[6]品〉《何義經》[7]第一

解經

[1]《法界聖凡水陸大齋法輪寶懺》卷2(卍續藏X74，955 c13～16 // Z 2B:2，435b1～4 // R129，869b1～4)：「一、《何義經》：阿難問持戒為何義。佛言：令不悔義。如是展轉問…歡悅、得喜、得止、得樂、得定、見知如實，得厭、無欲，即得解脫婬怒癡」

[2]《佛光阿含藏．中阿含（一）》(p.347，n.4)：

本經敍說佛告阿難：持戒者便得不悔，因不悔便得歡悅，乃至得解脫。如是此戒趣至彼岸。

※《增支部》(A. 10. 1. Kimatthiya 何義)。


一、序分

我聞如是：
一時，佛遊舍衛國，在勝林給孤獨園。
二、正宗分

阿難展轉問義，由持戒義至成解脫

1、別義

（1）持戒者便得不悔

爾時，尊者阿難則於晡時從燕[8]坐起，往詣佛所，稽首禮足，卻住一面，白曰：「世尊！持戒為何義？」
世尊答曰：「阿難！持戒者，令不悔[9]義。阿難！若有持戒者，便得不悔。」

（2）不悔者便得歡悅
復問：「世尊！不悔為何義？」
世尊答曰：「阿難！不悔者，令歡悅[10]義。阿難！若有不悔者，便得歡悅。」

（3）歡悅者，便得喜
復問：「世尊！歡悅為何義？」
世尊答曰：「阿難！歡悅者，令喜義。阿難！若有歡悅者，便得喜[11]。」
　　 （4）喜者，便得止身

復問：「世尊！喜為何義？」
世尊答曰：「阿難！喜者，令止義。阿難！若有喜者，便得止身[12]。」

     （5）止者，便得覺樂

復問：「世尊！止為何義？」
世尊答曰：「阿難！止者，令樂[13]義。阿難！若有止者，便得覺樂。」

（6）樂者，便得定心

復問：「世尊！樂為何義？」
世尊答曰：「阿難！樂者，令定義。阿難！若有樂者，便得定心。」[14]

（7）定者，便得見如實、知如真

復問：「世尊！定為何義？」
世尊答曰：「阿難！定者，令見如實．知如真[15]義。阿難！若有（485b）定者，便得見如實．知如真。」
　　 （8）見如實．知如真者，便得厭

復問：「世尊！見如實．知如真為何義？」
世尊答曰：「阿難！見如實．知如真者，令厭義。阿難！若有見如實．知如真者，便得厭[16]。」
     （9）厭者，便得無欲

復問：「世尊！厭為何義？」
世尊答曰：「阿難！厭者，令無欲[17]義。阿難！若有厭者，便得無欲。」
     （10）無欲者，便得解脫一切淫、怒、癡

復問：「世尊！無欲為何義？」
世尊答曰：「阿難！無欲者，令解脫[18]義。阿難！若有無欲者，便得解脫一切淫、怒、癡。

2、結義

是為，阿難！因持戒便得不悔，因不悔便得歡悅，因歡悅便得喜，因喜便得止，因止便得樂，因樂便得定。

阿難！多聞聖弟子因定便得見如實．知如真，因見如實．知如真，便得厭，因厭便得無欲，因無欲便得解脫，因解脫便知解脫：生已盡，梵行已立，所作已辦，不更受有，知如真。

※阿難！是為法法相益，法法相因。[19]

如是此戒趣至第一，謂度此岸，得至彼岸。」[20]
三、流通分

佛說如是，尊者阿難及諸比丘聞佛所說，歡喜奉行！
　　

《何義經》第一竟[21]（五百二十九字）[22]

《瑜伽師地論》卷28 (大正30，436a17～24) ：

問：何緣三學如是次第？

答：先於尸羅善清淨故便無憂悔；無憂悔故歡喜安樂；由有樂故心得正定；心得定故能如實知、能如實見；如實知見故能起厭；厭故離染；由離染故便得解脫；得解脫故證無所作究竟涅槃。

如是最初修習淨戒，漸次進趣，後證無作究竟涅槃。是故三學如是次第。

 

 

（四三）[23] 《中阿含》[24]〈習相應品〉《不思經》[25]第二（初一日誦）[26]

 

解經

[1]《法界聖凡水陸大齋法輪寶懺》卷2(卍續藏X74，955c16 // Z 2B:2，435b4 // R129，869b4)：「二、《不思經》：「持戒即得不悔，不須思量。」

[2]《佛光阿含藏．中阿含（一）》(p.351，n.4)：

本經敍說人不可但求不悔而不持戒，當持戒便得無悔，乃至得解脫。

※《增支部》(A. 10. 2. Cetanā 思)。

 

一、序分

我聞如是：
一時，佛遊舍衞國，在勝林給孤獨園。
二、正宗分

持戒者，但法自然即得不悔，不須思量；展轉乃至無欲者便得解脫

1、別義

爾時，世尊告曰：「

◎阿難！持戒者不應思：令我不悔。阿難！但法自然[27]，持戒者便得不悔。

◎阿難！有不悔者不應思：令我歡悅。阿難！但法自然，有不悔者便得歡悅。

◎阿難！有歡悅者不應思：令我喜。阿難！但法自然，有歡悅者便得喜。

◎阿難！有喜者不應思：令我止。阿難！但法自然，有喜者便得止身。

◎阿難！有止者不應思：令我樂。阿難！但法自然，有（485c）止者便得覺樂。

◎阿難！有樂者不應思：令我定。阿難！但法自然，有樂者便得定心。

◎阿難！有定者不應思：令我見如實．知如真。阿難！但法自然，有定者便得見如實．知如真。

◎阿難！有見如實．知如真者不應思：令我厭。阿難！但法自然，有見如實．知如真者便得厭。

◎阿難！有厭者不應思：令我無欲。阿難！但法自然，有厭者便得無欲。

◎阿難！有無欲者不應思：令我解脫。阿難！但法自然，有無欲者便得解脫一切淫、怒、癡。　　

2、結義

阿難！是為因持戒便得不悔，因不悔便得歡悅，因歡悅便得喜，因喜便得止，因止便得樂，因樂便得定心。阿難！多聞聖弟子有定心者便見如實．知如真，因見如實．知如真便得厭，因厭便得無欲，因無欲便得解脫，因解脫便知解脫：生已盡，梵行已立，所作已辦，不更受有，知如真。

※阿難！是為法法相益，法法相因，如是此戒趣至第一，謂度此岸，得至彼岸。」
三、流通分

佛說如是，尊者阿難及諸比丘聞佛所說，歡喜奉行！
　

　《不思經》第二竟[28]（四百五十字）[29]

一、序分

　　我聞如是：

一時，佛遊舍衞國，在勝林給孤獨園。
二、正宗分

（一）犯戒之弊：若比丘犯戒則害不悔；乃至無解脫便害涅槃

1、標宗

爾時，尊者舍梨子告諸比丘：「諸賢！若比丘犯戒，便害不悔、歡悅、喜、止、樂、定、見如實．知如真、厭、無欲、解脫；若無解脫，便害涅槃。

2、舉喻

3、合法

1、標宗

諸賢！若比丘持戒，便習不悔、歡悅、喜、止、樂、定、見如實．知如真、厭、無欲、解脫；若有解脫，便習涅槃。

2、舉喻

諸賢！猶如有樹，若不害根者，則莖、幹、心、節、枝、葉、華、實皆得成就。

3、合法

諸賢！當知比丘亦復如是，若有持戒，便習不悔、歡悅、喜、止、樂、定、見如實．知如真、厭、無欲、解脫；若有解脫，便習涅槃。」
三、流通分

一、序分

（490c）我聞如是：
一時，佛遊舍衞國，在勝林給孤獨園。
二、正宗分

明涅槃以解脫為習；乃至苦以老死為習

爾時，世尊告諸比丘：「

涅槃有習，非無習。

何謂涅槃習？

答曰：『解脫為習。』

解脫亦有習，非無習。

何謂解脫習？

答曰：『無欲為習。』

無欲亦有習，非無習。

何謂無欲習？

答曰：『厭為習。』

厭亦有習，非無習。

何謂厭習？

答曰：『見如實．知如真為習。』

見如實．知如真亦有習，非無習。

何謂見如實．知如真習？

答曰：『定為習。』

定亦有習，非無習。

何謂定習？

答曰：『樂為習。』

樂亦有習，非無習。

何謂樂習？

答曰：『止為習。』

止亦有習，非無習。

何謂止習？

答曰：『喜為習。』

喜亦有習，非無習。

何謂喜習？

答曰：『歡悅為習。』

歡悅亦有習，非無習。

何謂歡悅習？

答曰：『不悔為習。』

不悔亦有習，非無習。

何謂不悔習？

答曰：『護戒為習。』

護戒亦有習，非無習。

何謂護戒習？

答曰：『護諸根為習。』

護諸根亦有習，非無習。

何謂護諸根習？

答曰：『正念、正智為習。』

正念、正智亦有習，非無習。

何謂正念、正智習？

答曰：『正思惟為習。』

正思惟亦有習，非無習。

何謂正思惟習？

答曰：『信為習。』

信亦有習，非無習。

何謂信習？

答曰：『苦為習。』

苦亦有習，非無習。

何謂苦習？

答曰：『老死為習。』[135]

明老死以生為習…乃至行以無明為習

老死亦有習，非無習。

何謂老死習？

答曰：『生為習。』

生亦有習，非無習。

何謂生習？

答曰：『有為習。』

有亦有習，非無習。

何謂有習？

答曰：『受為習。』

受亦有習，非無習。

何謂受習？

答曰：『愛為習。』

愛亦有習，非無習。

何謂愛習？

答曰：『覺為習。』

覺亦有習，非無習。

何謂覺習？

答曰：『更樂為習。』

更樂亦有習，非無習。

何謂更樂習？

答曰：『六處為習。』

六處亦有習，非[136]無習。

何謂（491a）六處習？

答曰：『名色為習。』

名色亦有習，非無習。

何謂名色習？

答曰：『識為習。』

識亦有習，非無習。

何謂識習？

答曰：『行為習。』

行亦有習，非無習。

何謂行習？

答曰：『無明為習。』

是為緣無明行，緣行識，緣識名色，緣名色六處，緣六處更樂，緣更樂覺，緣覺愛，緣愛受，緣受有，緣有生，緣生老死，緣老死苦。

習苦，便有信；展轉乃至習解脫，便得涅槃。

習苦，便有信；習信，便有正思惟；習正思惟，便有正念、正智；習正念、正智，便有護諸根、護戒、不悔、歡悅、喜、止、樂、定、見如實．知如真、厭、無欲、解脫；習解脫，便得涅槃。」

三、流通分　　

佛說如是，彼諸比丘聞佛所說，歡喜奉行！

解經

[1]《法界聖凡水陸大齋法輪寶懺》卷2(卍續藏X74，956a16～22 // Z 2B:2，435c10～16 // R129，870a10～16)：「十五、《彌醯經》：彌醯侍佛，辭佛住靜處求斷，反起欲、恚、害，三害念；乃歸佛所；佛為說心解脫未熟，欲令熟者有五習法：一與善知識俱；二護戒威儀；三說聖法；四精進斷惡修善；五觀興衰法。又修四法：惡露斷欲、慈斷恚、出入息斷亂、無常斷慢，而皆以與善知識俱，乃得成後四事及四法。」

[2]《佛光阿含藏．中阿含（一）》(p.381，n.3)：

本經敍說尊者彌醯侍佛，後辭佛往林靜處修行，反起三惡念。佛知其心解脫未熟故，為說五習法：(一)善知識與善知識相交，(二)受持學戒，(三)談論戒定慧等法義，(四)常行精進，(五)修智慧善觀。其間，皆當與善知識俱，而修惡露、慈、息出息入、無常想等四法。

※《增支部》(A. 9. 3. Meghiya 彌醯)、《小部‧自說經》(Ud. 4. 1)。

 

一、序分

我聞如是：
一時，佛遊摩竭陀國[143]，在闍鬪[144]村莽木[木*奈][145]窟。
二、正宗分

（一）佛之侍者彌醯，巧遇靜處，興欲往住而求斷

爾時，尊者彌醯[146]為奉侍者。

於是，尊者彌醯過夜平旦，著衣持鉢，入闍鬪村而行乞食。乞食已竟，往至金鞞[147]河邊，見地平正，名好[木*奈]林；金鞞河水極妙可樂，（491b）清泉徐流，冷暖和適。見已歡喜，便作是念：此地平正，名好[木*奈]林；金鞞河水極妙可樂，清泉徐流，冷暖和適。若族姓子欲學斷者，當於此處[148]；我亦有所斷，寧可在此靜處學斷耶？
　（二）彌醯求辭欲往住靜處，佛再三勸說慰留，彌醯辭意堅持，佛默言應允

1、初表求去

於是，彌醯食訖，中後舉[149]衣鉢已，澡洗手足，以尼師檀[150]著於肩上，往詣佛所，稽首禮足，卻住一面，白曰：「世尊！我今平旦著衣持鉢，入闍鬪村而行乞食。乞食已竟，往至金鞞河邊，見地平正，名好林[木*奈]；金鞞河水極妙可樂，清泉徐流，冷暖和適。我見喜已，便作是念：此地平正，名好[木*奈]林；金鞞河水極妙可樂，清泉徐流，冷暖和適。若族姓子欲學斷者，當於此處；我亦有所斷，寧可在此靜處學斷耶？世尊！我今欲往至彼[木*奈]林靜處學斷。」

2、佛勸留小住

爾時，世尊告曰：「彌醯！汝今知不？我獨無人，無有侍者，汝可小住，須比丘來為吾侍者，汝便可去，至彼[木*奈]林靜處而學。」

3、彌醯再三求辭

尊者彌醯乃至再三白曰：「世尊！我今欲往至彼[木*奈]林靜[151]處學斷。」

4、佛再三慰留

世尊亦復再三告曰：「彌醯！汝今知不？我獨無人，無有侍者，汝可小住，須比丘來為吾侍者，汝便可去，至彼[木*奈]林 *靜處而學。」

5、推委佛為無所斷，彌醯堅持辭意

彌醯復白曰：「世尊無為無作，亦無所觀。世尊！我有為有作而有所觀。世尊！我至彼[木*奈]林靜處學斷。」

6、佛不再勸止，應允彌醯所欲

世尊告曰：「！汝欲求斷者，我復何言？彌醯汝去，隨意所欲。」

（三）入林求斷起三惡念

◎於是，尊者彌醯聞佛所說，善受、善持而善誦習，卽禮佛足，繞三匝而去。詣彼[木*奈] 林，入林中已，至一樹下，敷尼師*檀，結跏[152]趺坐。

◎尊者彌醯住[木*奈]林中便生三惡不善之念：欲念、恚念及與害念。[153]彼由此故便念世                 尊。

 

（四）尊者返歸佛所明所起惡尋；佛知其心解脫未熟故，為說五習法、復修四法；明與善士俱，得成諸善法，乃至涅槃

1、尊者返歸佛所明所起惡尋

◎於是，彌醯則於晡時從 *燕[154]坐起，往詣佛所，稽首禮足，卻住一面，白曰：「世尊！我至[木*奈]林，於靜處坐便生三惡不善之念：欲念、恚念及與害念。我由此故便念世尊。」

2、佛為說五習法

世尊告曰：「彌醯！心解脫未熟，欲令熟者有五習法。云何為五？

（1）與善知識相交和合

彌醯！比丘者，自善知識與善知識俱[155]，善知識共和合。彌醯！心解脫未熟，欲令熟者，是謂第一習法。

（2） 修習、守護禁戒律儀

復次，彌醯！比丘者，修習禁戒，守護從解脫，又復善攝（491c）威儀禮節，見纎芥罪，常懷畏怖，受持學戒。彌醯！心解脫未熟，欲令熟者，是謂第二習法。

（3）談論皆是說聖有義之法

復次，彌醯！比丘者，謂所可說聖有義，令心柔軟，使心無蓋，謂說戒、說定、說慧、說解脫、說解脫知見、說漸損、說不樂聚會、說少欲、說知足、說斷、說無欲、說滅、說 *燕坐、說緣起，得如是比沙門所說，具得，易不難得。彌醯！心解脫未熟，欲令熟者，是謂第三習法。

（4）常行精進斷惡修善

復次，彌醯！比丘者，常行精進，斷惡不善，修諸善法，恒自起意，專一堅固，為諸善本，不捨方便。彌醯！心解脫未熟，欲令熟者，是謂第四習法。

（5）修行智慧，善觀興衰法

復次，彌醯！比丘者，修行智慧，觀興衰法，得如是智，聖慧明達，分別曉了，以正盡苦。彌醯！心解脫未熟，欲令熟者，是謂第五習法。

3、復修四法
彼有此五習法已，復修四法。云何為四？

◎修惡露[156]，令斷欲；

◎修慈，令斷恚；

◎修息出息入[157]，令斷亂念；

◎修無常想，令斷我慢。 [158]

4、明與善知識俱，必修必行成就四習法、四事，乃至涅槃

（1）必修必行成就四習法

Ａ、戒

彌醯！若比丘自善知識與善知識俱，善知識共和合，當知必修習禁戒，守護從解脫，又復善攝威儀禮節，見纎芥罪，常懷畏怖，受持學戒。

Ｂ、說聖有義語

彌醯！若比丘自善知識與善知識俱，善知識共和合，當知必得所可說聖有義，令心柔軟，使心無蓋，謂說戒、說定、說慧、說解脫、說解脫知見、說漸損、說不樂聚會、說少欲、說知足、說斷、說無欲、說滅、說 *燕坐、說緣起，得如是比沙門所說，具得，易不難得。

Ｃ、精進

彌醯！若比丘自善知識與善知識俱，善知識共和合，當知必行精進，斷惡不善，修諸善法，恒自起意，專一堅固，為諸善本，（492a）不捨方便。

Ｄ、智慧

彌醯！若比丘自善知識與善知識俱，善知識共和合，當知必行智慧，觀興衰法，得如此智，聖慧明達，分別曉了，以正盡苦。

（2）必修四事；得無常想者得無我想→斷我慢→得涅槃

◎彌醯！若比丘自善知識與善知識俱，善知識共和合，當知必修惡露，令斷欲；修慈，令斷恚；修息出息入，令斷亂念；修無常想，令斷我慢。

◎彌醯！若比丘得無常想者，必得無我想。

彌醯！若比丘得無我想者，便於現法斷一切我慢，得息、滅、盡、無為、涅槃。[159]
三、流通分

佛說如是，尊者彌醯及諸比丘聞佛所說，歡喜奉行！
　

《彌醯經》第十五竟[160]（千三百二十三字）[161]

◎若比丘得無常想者，必得無我想。

若比丘得無我想者，便於現法斷一切我慢，得息、滅、盡、無為、涅槃。」
